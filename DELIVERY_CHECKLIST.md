## 📝 使用前准备

### 用户需要准备
1. **OpenAI API Key**（或其他兼容 API）
   - 用于 TTS 功能
   - 用于 AI 翻译和解释

2. **Obsidian 最低版本**
   - 需要 0.15.0 或更高版本

3. **网络连接**
   - 稳定的网络连接访问 API

### 配置步骤
1. 在 Obsidian 中启用插件
2. 进入设置页面
3. 填写 TTS API 配置
4. 填写 AI API 配置
5. 自定义提示词（可选）
6. 开始使用
