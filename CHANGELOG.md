# 更新日志

## [1.0.0] - 2025-12-15

### ✨ 核心功能

#### 🔊 智能朗读（TTS）
- 双模式支持：浏览器内置 TTS / API 服务（OpenAI TTS）
- 语速和音调调节
- 自动检测系统可用语音

#### 🌐 AI 翻译
- 流式输出翻译结果
- 自定义目标语言
- 一键复制或替换原文

#### 💡 AI 搜索（解释）
- 流式输出智能解释
- 自定义输出语言
- 可追加解释到笔记末尾

#### 📄 智能总结
- 流式输出总结内容
- 刷新/插入/复制操作
- 自定义输出语言

#### 📖 识词功能
- 智能识别单词、短语或成语
- 音标/拼音展示
- 浏览器 TTS 发音
- AI 生成例句

### 🎨 用户界面
- 选中文本自动显示悬浮工具栏
- 可拖动菜单位置
- 响应式设计（亮色/暗色主题）
- 图标 + 文字标签

### ⚙️ 配置选项
- 功能开关（独立开启/关闭各功能）
- TTS 提供商选择和参数配置
- AI API 配置（支持 OpenAI/Ollama 等）
- 自定义提示词模板
