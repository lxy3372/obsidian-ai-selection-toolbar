# 🎬 使用演示指南

## 📖 演示场景

本指南提供完整的使用演示场景，帮助你快速了解插件的所有功能。

## 🎯 场景 1: 朗读英文段落

### 场景描述
你在阅读一篇英文文章，想要听听某段话的发音。

### 操作步骤

1. **打开笔记**
   ```
   在 Obsidian 中打开任意包含英文文本的笔记
   ```

2. **选中文本**
   ```
   选中这段文本:
   "Artificial intelligence is transforming the way we work and live."
   ```

3. **等待菜单出现**
   ```
   等待 300ms，悬浮菜单自动出现在选区上方
   ```

4. **点击朗读按钮**
   ```
   点击 🔊 图标
   ```

5. **听音频播放**
   ```
   - 按钮变为"播放中"状态
   - 听到 TTS 语音朗读
   - 可再次点击停止播放
   ```

### 预期效果
- ✅ 清晰的英文语音播放
- ✅ 播放状态实时反馈
- ✅ 可随时停止

## 🌐 场景 2: 翻译中文为英文

### 场景描述
你写了一段中文，需要快速翻译成英文。

### 操作步骤

1. **选中中文文本**
   ```
   选中这段文本:
   "人工智能正在深刻改变我们的工作和生活方式，它不仅提高了效率，还开启了全新的可能性。"
   ```

2. **点击翻译按钮**
   ```
   点击 🌐 图标
   ```

3. **查看翻译结果**
   ```
   结果面板展开，显示:
   "Artificial intelligence is profoundly changing our work and lifestyle. 
   It not only improves efficiency but also opens up new possibilities."
   ```

4. **使用翻译结果**
   
   **选项 A: 复制**
   ```
   点击 📋 复制图标
   → 翻译结果已复制到剪贴板
   → 可粘贴到任何地方
   ```

   **选项 B: 替换**
   ```
   点击 🔄 替换图标
   → 原文被译文替换
   → 笔记更新完成
   ```

### 预期效果
- ✅ 准确的翻译结果
- ✅ 保持原文意思
- ✅ 自然流畅的表达

## 💡 场景 3: 解释专业术语

### 场景描述
你在笔记中遇到一个专业术语，需要快速了解其含义。

### 操作步骤

1. **选中术语**
   ```
   选中: "Obsidian"
   ```

2. **点击解释按钮**
   ```
   点击 💡 图标
   ```

3. **阅读解释**
   ```
   结果面板显示:
   "Obsidian 是一款基于 Markdown 的知识管理软件，
   采用本地优先的理念，所有笔记存储在本地文件系统中。
   它以双向链接和图谱视图为特色，帮助用户构建个人知识网络。"
   ```

4. **保存解释**
   
   **选项 A: 复制**
   ```
   点击 📋 复制图标
   → 解释已复制
   ```

   **选项 B: 追加到笔记**
   ```
   点击 ➕ 追加图标
   → 解释自动插入到选中文字之后
   → 格式: "\n\n[解释内容]"
   ```

### 预期效果
- ✅ 清晰易懂的解释
- ✅ 简洁不冗长
- ✅ 适合初学者理解

## 🎯 最佳实践

### DO（推荐做法）

✅ **选中合适长度的文本**
- TTS: 一段话（50-200 字）
- 翻译: 一段或多段（100-500 字）
- 解释: 单词或短语（1-20 字）

✅ **等待菜单完全出现**
- 不要急于点击
- 确保防抖时间结束

✅ **合理使用 API**
- 避免频繁调用
- 注意 API 费用
- 缓存常用结果

### DON'T（避免的做法）

❌ **不要选中过长文本**
- 避免一次选中数千字
- 可能导致 API 超时

❌ **不要频繁切换选择**
- 等待前一次操作完成
- 避免触发防抖

❌ **不要在移动端使用**
- 目前主要为桌面端优化
- 移动端体验可能不佳
